# 📱 Programação Mobile — Portfólio de Aplicativos

Este repositório reúne os aplicativos desenvolvidos durante a disciplina de **Programação Mobile**, utilizando a plataforma **MIT App Inventor** para criação de apps Android.

Os projetos aqui presentes foram criados com o objetivo de praticar lógica de programação, construção de interfaces e desenvolvimento de aplicações funcionais para dispositivos móveis.

Ao longo do repositório é possível acompanhar desde aplicativos simples até projetos mais completos com múltiplas telas e interação com o usuário.

---

## 📌 Finalidade do Repositório

Este portfólio tem como objetivo:

* Registrar os aplicativos desenvolvidos em aula
* Praticar lógica de programação mobile
* Trabalhar com eventos e componentes visuais
* Desenvolver aplicativos funcionais
* Organizar projetos da disciplina
